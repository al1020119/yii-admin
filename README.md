# Yii后台管理系统

> 可作为Yii实现后台管理的基础框架

+ 此项目为个人在公司做的一个后台管理项目，经过处理后的代码！

### 支持功能

1. 登录，退出
2. 用户管理
3. 用户级别
4. 用户创建，修改，删除，控制权限
5. 三级联动
6. 搜索
7. 分页
8. 仪表盘
9. 操作日志


> 下载项目后注意composer install

+ 如有任何疑问，请以邮件的方式联系我，相互学习交流！
